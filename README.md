# 📋 Lista de Tarefas em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## 🚀 Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Validação de tarefa existente
- Encerramento do sistema
- Menu interativo com loop

## 🛠️ Tecnologias utilizadas

- Python
- VS Code

## 🎯 Objetivo

Praticar:
- Listas
- Loops
- Estruturas condicionais
- Manipulação de dados
- Lógica de programação

## ▶️ Como executar

1. Instale o Python
2. Execute o arquivo:

python lista_tarefas.py
